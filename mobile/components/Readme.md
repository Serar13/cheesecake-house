# Components Directory

This directory contains reusable UI components for the React Native mobile application:

- `CakeCard.js` - A visually appealing card to display a cheesecake item.
- `CartItem.js` - List item component used in the Cart screen.
- `Button.js` - Custom themed button matching the Cheesecake House aesthetics.
- `LocationCard.js` - Card showcasing store address, schedule, and phone number.
